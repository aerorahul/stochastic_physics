Detailed Description

The module 'get_stochy_pattern_mod' contains the subroutines to retrieve the random pattern in the cubed-sphere grid.

subroutine, public get_stochy_pattern_mod::dump_patterns ( character*120  sfile)

The subroutine 'dump_patterns' writes out the speherical harmonics to a file, controlled by FHSTOCH.

Only the active patterns are written out
